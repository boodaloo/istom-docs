#### Настройка прав доступа.

В программе есть возможность распределить роли для всех своих сотрудников, это нужно для того чтобы, определить какими полномочиями будет обладать той или иной пользователь. Для этого необходимы группы доступа.

Группы доступа по умолчанию существуют такие:

1) Администратор - это полный доступ ко всем зарплатам всех врачей, всей финансовой информации и любой имеющейся в iStom. (По умолчанию Администратор имеет право вносить\\изменять\\удалять данные почти во всех разделах программы)

2) Ресепшен, Врач, Пользователь -  эти группы доступа дают возможность доступа ко всем отмеченным галочкой разделам соответствующим должности сотрудника. Раздел зарплатные ведомости может видеть только тот, под кем мы авторизовались. Например Иванов И.И., он врач, и значит, что он будет видеть только свою зарплату и счета.

Для удобства представлена возможность создания\\редактирования\\удаления групп доступа, а также присутствует функция подгрупп, которые вы также можете настроить по своему усмотрению.
