# :material-calendar-clock: Расписание

Модуль расписания - ключевой инструмент для управления записью пациентов.

---

## Основные возможности

- Запись пациентов на прием
- Настройка рабочего времени клиники
- Индивидуальные графики врачей
- Цветовая индикация статусов приема
- Быстрый поиск свободного времени
