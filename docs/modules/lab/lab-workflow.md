## Как организована зуботехническая лаборатория (ЗТЛ) в iStom

#### ?

В программе есть возможность формировать прайс для ЗТЛ, но только тогда, когда добавлены техники и лаборатории. Их можно добавить в разделе \"Заказ-наряд\".

## Разработана ли версия для лаборатории (лаборантов)

#### Разработана ли версия для лаборатории (лаборантов)?

Да, у наших партнеров разработана версия для ЗТЛ. Программа ZTLsoft интегрируется с iStom.

Демо версия ZTLsoft:

<http://demo.ztlsoft.ru/demo.zip>

Программа устанавливается у администратора или управляющего лабораторией.

Программа позволяет вести весь учет от и до.

Для техникам будет разработано мобильное приложение.

Интеграция с iStom уже в процессе.

По всем вопросам касательно программы ZTLsoft вы можете позвонить по номеру  +74992132390. Сайт программы: https://ztlsoft.ru/

## ПЛ- ЗТЛ

#### Возможно ли вводить отдельные прайс-листы для лабораторий и техников?

Да, такая возможность присутствует. Можно добавить неограниченное число прайс-листов для отдельных техников и лабораторий.
