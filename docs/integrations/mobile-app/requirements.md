#### На каком телефоне можно поставить iStom Mobile?

Для того, чтобы установить мобильную версию программы вам необходимо, чтобы версия Android на вашем телефоне была не ниже android 4.4 (Версию Андроид можно посмотреть в настройках телефона, пункт в настройках разных телефонов может называться по разному, например: \"Система\", \"Об устройстве\", \"О телефоне\" и т.п.)
