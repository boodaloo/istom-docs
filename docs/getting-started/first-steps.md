#### Как компьютеры работают локально с программой IStom?

Через компьютер с базой данных, на котором существует статический IP, идет подключение всех дочерних компьютеров. IP не меняется при подключении. Также существует динамический адрес, который меняется при подключении. Мы не несем ответственность за стабильность работы и доступ к базе таким методом.

Статический IP платно подключается  у провайдера, если компьютеры находятся в разных зданиях. Если компьютеры находятся в одном здании - то IP подключается бесплатно и работает в локальной сети.
