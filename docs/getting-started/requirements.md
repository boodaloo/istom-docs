Минимальные технические требования

• Intel Pentium 4 или более быстрые x86 или х64-разрядный процессор.

• 1440x900 - минмиальное разрешение экрана.

• Клавиатура и мышь.

• 100 MB места на жестком диске для установки iStom на компьютер. 5 ГБ места на жестком диске для установки локальной версии на серверном компьютере.

• Microsoft Windows 7 Service Pack 1 (32 bit and 64 bit), Windows 8 or Windows 10.

• 1GB of RAM (2GB of RAM recommended).

## есть ли версия для Mac

#### Есть ли версия программы для Mac?

Работа с IStom на Mac возможна при помощи программ VirtualBox, Parallels или VMWare для создания виртуальных машин.

1 вариант.

На макбуке можно запустить Windows через утилиты, чтобы при запуске компьютера была доступна для выбора одна из двух операционных систем: Mac и Windows.

2 вариант.

Приобрести эмулятор Parallels, VmWare либо другой. Их также можно скачать в России бесплатно.

Это сторона работы системного администратора клиники. Вы подготавливаете компьютер для работы на Windows, после чего мы устанавливаем программу на ваш макбук.
